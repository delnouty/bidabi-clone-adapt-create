# Structure attendue du jeu de données 

Les étudiants doivent créer un jeu de données d’images organisé comme suit :
```
data/
├── train/
│   ├── class_1/
│   │   ├── image1.jpg
│   │   ├── image2.jpg
│   ├── class_2/
│   │   ├── image1.jpg
│   │   ├── image2.jpg
|   |   |...
├── val/
│   ├── class_1/
│   ├── class_2/
|   |...
```

- Les noms de classes peuvent être choisis librement.
- Les images doivent être en format `.jpg` ou `.png`.
- Le dossier `test/` est optionnel mais recommandé pour l’évaluation.
